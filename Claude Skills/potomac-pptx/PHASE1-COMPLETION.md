## 📋 USAGE GUIDE

### **Generate Basic Presentation**
```bash
cd skills/potomac-pptx
node scripts/generate-potomac-presentation.js
```

### **Custom Presentation**
```bash
node scripts/generate-potomac-presentation.js \
  --title "YOUR PRESENTATION TITLE" \
  --subtitle "Custom Subtitle" \
  --output "custom-presentation.pptx" \
  --palette INVESTMENT
```

### **Available Palettes**
- `STANDARD` - Default Potomac yellow and gray
- `DARK` - Premium dark background theme
- `INVESTMENT` - Turquoise accent for Investment Strategies
- `FUNDS` - Turquoise primary for Potomac Funds

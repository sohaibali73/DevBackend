## 🎬 ANIMATION STANDARDS

### General Guidelines
1. **Timing:** All skill animations should be between 300ms and 700ms total duration
2. **Easing:** Use `ease-out-cubic` for all transitions
3. **Color Scheme:** Maintain brand colors: Potomac Yellow `#FEC00F`, Dark Gray `#212121`
4. **Accessibility:** Provide reduced motion alternatives
5. **State Indicators:**
   - 🟢 Active: Solid icon + pulsing border
   - 🟡 Loading: Animation in progress
   - 🔴 Error: Shake animation + red border
   - ⚪ Disabled: Grayscale + 50% opacity

### Animation Types
| Animation Class | Use Case | Timing |
|----------------|----------|--------|
| `skill-fade` | Standard appearance | 300ms |
| `skill-slide-left` | Data & document skills | 400ms |
| `skill-slide-up` | Presentation & UI skills | 350ms |
| `skill-scale` | Analysis & quant skills | 450ms |
| `skill-pulse` | Loading states | 600ms |
